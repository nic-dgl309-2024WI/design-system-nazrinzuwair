Flex Layout

Flex layout is a CSS layout model that allows you to create flexible and responsive webpage designs with ease. ith flex layout, you can control the alignment, distribution, and order of elements within a container, making it ideal for creating complex and dynamic layouts.

Max-width-Desktop

The max-width: 1200px; CSS property sets the maximum width of an element to 1200 pixels. This means that the element cannot exceed this specified width, even if its content or parent container is wider.

Utility Class Name: .l-max-width-desktop

Value: max-width: 1200px;

Max-width-Mobile

The CSS property `max-width: 100%;` ensures that an element does not exceed the width of its containing parent element, maintaining responsiveness. This is commonly used in responsive web design to prevent elements from overflowing and to adapt to different screen sizes. By setting the maximum width to 100%, elements can resize proportionally and maintain a consistent layout across various devices.

Utility Class Name: .max-width: 100%;

Value: max-width: 100%;

Full-width-Desktop

Setting `width: 100%;` specifically for desktop ensures that the element spans the entire width of its containing parent only on desktop devices. This approach allows for a responsive design where the element adapts its width based on the viewport size, maximizing available space on larger screens while maintaining readability and usability.

Utility Class Name: .l-full-width

Value: width: 100%;

Full-width-mobile

For mobile, setting `width: 100%;` ensures that the element occupies the full width of the viewport, optimizing the display for smaller screens. This approach maximizes the use of limited screen real estate, allowing content to be easily viewed and interacted with on mobile devices.

Utility Class Name: .l-full-width

Value: width: 100%;
